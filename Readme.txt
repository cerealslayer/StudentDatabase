A Database designd to Enroll students in Programs, registar in courses and can also show a proffesors teaching profile

In the model the "Grade", "StudentProgram" and "CourseProffesor" serve as mediators between their connected classes.
The Student,Proffesor and Program are all normalized holding only primary keys as determinents or also have a forigen key for another table
The course Table has been denormalized, the semester table has been seperated from course table for better simplicity and understanding
